# Secure Password Generator (Java)

A lightweight Java console application that generates strong, cryptographically secure passwords.

## Features
* Uses `SecureRandom` for high-quality randomness.
* Includes uppercase, lowercase, numbers, and special characters.
* Clean and modular code.
